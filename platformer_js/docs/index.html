Getting Started

The game works directly in the browser without any build step:

# Clone the repository
git clone https://github.com/yourusername/platformer_js.git

# Open in browser
open index.html

# Or start a local server
npx serve

Development

For development with hot reload:

# Install dependencies
npm install

# Start dev server
npm run dev

# Run tests
npm test

Controls

  • Arrow Keys - Move and jump
  • Space - Shoot (after collecting power-up)
